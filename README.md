# Задание 1

### Текст задачи

Дерево содержит целые числа. Заменить минимальную цифру в каждом числе
на заданную.

### Алгоритм решения

1. Определяем тип бинарного дерева BinaryTree
2. Создаем генератор случайных чисел rnd
3. Создаем рекурсивную функцию generateRandomTree для генерации случайного дерева с заданной глубиной
4. Создаём функцию replaceMinDigit, которая заменяет минимальную цифру в числе на заданную
5. Создаём рекурсивную функцию mapTree, которая применяет функцию ко всем узлам дерева
6. Создаём рекурсивную функцию printTree, которая выводит дерево в консоль
7. В main:
- Устанавливаем глубину дерева = 4
- Генерируем случайное дерево вызовом generateRandomTree depth и сохраняем в root
- Выводим исходное дерево
- Создаём новое дерево newRoot с помощью mapTree, применяя к каждому узлу функцию replaceMinDigit x 9 (заменяем минимальную цифру на 9)
- Выводим новое дерево

# Задание 2

### Текст задачи

Дерево содержит строки. Сколько узлов дерева содержит заданный символ?

### Алгоритм решения

1. Определяем тип BinaryTree — узел содержит целое число, левое и правое поддеревья, либо пустое дерево
2. Создаем генератор случайных чисел rnd
3. Создаем рекурсивную функцию generateRandomTree depth:
   - Если depth = 0 возвращаем Empty
   - Иначе:
     - Генерируем случайное число от -999 до 999
     - С вероятностью 70% создаем левое поддерево вызовом generateRandomTree (depth - 1), иначе Empty
     - С вероятностью 70% создаем правое поддерево вызовом generateRandomTree (depth - 1), иначе Empty
     - Возвращаем Node(число, левое, правое)

4. Создаем функцию replaceMinDigit number replacement`:
   - Берем абсолютное значение числа
   - Если число = 0 → возвращаем replacement
   - Иначе:
     - Находим минимальную цифру в числе
     - Заменяем все вхождения минимальной цифры на replacement
     - Собираем новое число
     - Возвращаем число с сохранением знака

5. Создаем рекурсивную функцию mapTree f tree:
   - Если дерево пустое → возвращаем Empty
   - Если дерево узел → возвращаем Node(f data, mapTree f left, mapTree f right)

6. Создаем рекурсивную функцию printTree indent tree:
   - Если дерево пустое → ничего не делаем
   - Если дерево узел → выводим отступ и значение, затем рекурсивно выводим левое и правое поддеревья с увеличенным отступом

7. В `main`:
   - Устанавливаем depth = 4
   - Генерируем дерево: root = generateRandomTree depth
   - Выводим "Исходное дерево:" и печатаем root
   - Создаем newRoot = mapTree (fun x -> replaceMinDigit x 9) root
   - Выводим "Новое дерево
